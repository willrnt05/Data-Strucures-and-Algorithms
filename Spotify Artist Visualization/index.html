Data Source:
The data for this project was generated from the spotify web API using the Spotipy Python library. Specifically, this data set was generated by recursively visiting each artist up to 4 steps removed from a set of starting artists. Starting artists were taken from: the billboard top 125 artists of all time, the billboard top artists of the year 1981-2015, and the billboard top artists of 2019.

Note: The implemention of Dijkstra's algorithm used in this project does use utilize a priority queue and as a result runs much slower than the other algorithms.
